YouTube-Parser

This service allows you to dynamically analyze an YouTube account — to track daily statistics changes, determine the best topics, choose the most successful bloggers and outsiders. The account is linked to the platform without gaining access to channel, it is enough to send a link.
The service could work with any number of accounts. Additionally, we’ve made it easy to send tasks / check / communicate with users using an admin dashboard. The functionality is convenient for the YT-schools. Users get messages with home tasks in TG-bot, but on request it might be developed a course platform with the highest level of video protection, an optimize interface for your user and administrator, interactive home tasks and much more.

Service development technology:
The service is created in the Django framework. An user’s YouTube-channels are checked with celery (on redis) regularly (once per day). Channel and video metrics are based for the last 7 days. All the information is considered, converted and displayed on appropriate pages. Administrators send and check home tasks in the admin dashboard. If a user hasn’t been active for more then 6 months (this period could be changed) in the bot or on YouTube, he is excluded from the list of daily checks. The bot is created in telebot and introduced directly into Django.
